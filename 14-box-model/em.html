Computed value of the font-size property, of the element on which it is used.
In this case it is inheriting font-size from the body and then it is gonna double it, and that is the computed value for this p element.

And the same process is happening for the rest of CSS properties we've used here; margin-right, border-radius.

Important: Please note that here strong element is inheriting its font-size from its p parent. Therefore when we are specifying its value it is relative to its parent (which is already doubled 14 * 2). Thus it is gonna be 24 * 2
